# Laptop Listing Page (React)

This is a responsive laptop listing page built using React. The project demonstrates a clean component-based architecture with reusable UI components and responsiveness.

## Components Overview

Header – Displays branding with styled UI

ProductImage – Handles image gallery and navigation

Specifications – Shows laptop technical details

SellerInfo – Displays seller profile and rating

LaptopCard – Main container combining all sections

LoadingSkeleton – Placeholder UI during data loading

## Tech Stack
React (Functional Components)

CSS3 (Custom styling per component)

JavaScript (ES6+)

Font Awesome (Icons)

## Responsive Design

The layout is fully responsive and optimized for mobile, tablet, and desktop screens using a mobile-first approach.
